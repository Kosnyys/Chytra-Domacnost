# Chytra-Domacnost
Projekt je o vytvoření jednoduché chytré domácnosti, která ukáže, jak funguje automatické ovládání různých zařízení a senzorů. Bude možné sledovat, co senzory měří, a ovládat připojená zařízení. Projekt pomáhá naučit se, jak tyto technologie fungují a jak mohou usnadnit každodenní život.
